What is flash-install?

flash-install is a fast, drop-in replacement for npm install, focused on drastically speeding up Node.js dependency installation through deterministic caching, parallel operations, and .flashpack archive snapshotting.

Installation

npm install -g @flash-install/cli

Basic Usage

# Standard installation
flash-install

# Create a snapshot
flash-install snapshot

# Restore from a snapshot
flash-install restore

# Clean up
flash-install clean

# Clean only node_modules (preserves snapshot)
flash-install clean-modules

# Clean only snapshot (preserves node_modules)
flash-install clean-snapshot

Advanced Features

  • Selective Cleaning: Clean only node_modules or only snapshots
  • Cache Management: Optimize and verify your global cache
  • Plugin System: Extend functionality with custom plugins
  • Offline Mode: Work without internet connection
  • Custom Snapshot Formats: Choose between different compression formats
  • Checksum Validation: Verify package integrity against npm registry checksums
  • Snapshot Fingerprinting: Auto-invalidate snapshots when lockfiles change
